# Phase1（default）改行指定

改行を入れたい位置に `\n` を挿入して返してください。
`\n` なし = 現状維持（「、」で自動分割）

---

## active（01〜15）

```
01: 予定や習慣を、少しずつ自分色に染めていきたい
02: レストランのメニューなどで迷っている時は、すべて決めてあげたい
03: 私が選んだ服を少し照れながらも、身に纏う姿がいとおしい
04: 相手の隠し事を少しずつ逃げ道を塞いで\n自ら打ち明けるのを静かに待ちたい
05: ちょっとした意地悪な言葉をかけて、戸惑う瞳をじっくり観察したい
06: 二人だけの「特別なルール」を作り、それを守らせることに密かな喜びを感じる
07: 私がいなければダメになってしまうくらい、深く依存されることに惹かれる
08: 甘えを許さず、けれど逃げ場のない正論で諭す時間が好き
09: 欲しがっているものをすぐには与えず、甘い焦らしを楽しみたい
10: 指先や声のトーンひとつで、感情が揺れ動くのを感じたい
11: 相手の肌に、私だけが知っている「秘密の印」をそっと残したい
12: 私に対してのみ相手が言葉遣いを崩さず、丁寧な態度で接してくるのが好き
13: あえて情報を与えず、私の言葉だけを信じてついてきてほしい
14: 普段は気丈な相手が、私にだけは素直に折れて頼ってくる姿に強く惹かれる
15: 私が描いた完璧なシナリオの中で、心地よさそうに振る舞う姿を見るのが好き
```

## passive（16〜30）

```
16: 休日の予定も着る服も、すべて誰かに「これがいい」と決めてほしい
17: 「ここで待っていて」と言われたら、その言葉に縛られて静かに待っていたい
18: 自分で決断するより、少し強引なくらいに手を引かれる方が安心する
19: 失敗した時は優しく慰められるより、静かに厳しく諭されたい
20: スケジュールや余裕をすべて他人に預け、自分の意思を手放すことに憧れる
21: 自分が誰かの「特別」であるより、絶対的に必要な「一部」として扱われたい
22: 「いい子だね」と静かに認められるためなら、少しの無理も心地よく感じる
23: 相手の期待を裏切ってしまった時、冷たい態度で突き放されると逆に目が離せなくなる
24: 限界まで焦らされ、自分から「お願い」をして初めて与えられることに喜びを感じる
25: 満員電車や狭い空間など、身動きが取れず「どうしようもない状況」に身を任せる感覚が好き
26: 目を隠されたり情報を遮断されたりして、すべてを委ねる時の不安と安堵
27: 重みのあるアクセサリーやタイトな服など、少し窮屈なものを好んで身につける
28: 相手の温度のない視線や、時折見せる無関心な態度にどうしても惹かれてしまう
29: 冷たく突き放された後、不意に見せられる優しい執着に抗えない
30: 自分の弱さや秘密をすべて握られ、もう後戻りできない状況に密かな憧れがある
```

## observer（31〜40）

```
31: 完全に一人の空間より、誰かの「視線」を感じるかもしれない緊張感が好き
32: 窓ガラスや鏡越しに自分たちの姿を、映画のワンシーンのように客観的に眺めたい
33: 人前では真面目な関係を装い、二人きりになった瞬間に空気が反転するのが好き
34: 他の人がいる前で、二人にしか分からない視線や秘密の合図を交わしたい
35: 他人のスマートフォンや手帳など、見てはいけない領域に触れる瞬間のスリル
36: 日常の自分とは全く違う、もう一つの「顔」を演じることに密かな快感を覚える
37: 誰にも言えない秘密の取り決めを交わし、その見えない糸に縛られる関係に憧れる
38: 感情をぶつけ合うよりも、お互いの出方を探り合うような静かな頭脳戦が好き
39: 相手の一番脆い部分を握り、それを盾に静かに微笑む関係が心地よい
40: 一線を越えるか越えないかの、曖昧でヒリヒリとした境界線を長く漂っていたい
```

## sensory（41〜50）

```
41: 静かな廊下に響くヒールの音や足音に、理由のない緊張と魅力を感じる
42: 部屋の鍵が「カチャリ」と閉まる音を聞くと、特別な時間が始まるという高揚感がある
43: 体温を持たない冷たい金属や、滑らかなシルクの質感に触れるのが好きだ
44: 相手の呼吸や衣擦れの音だけがリアルに感じられる、薄暗い空間に惹かれる
45: 首元を軽く覆うタートルネックや、手首を飾るブレスレットの重みが好きだ
46: 予測できないタイミングで、不意に背後から距離を詰められると心が跳ねる
47: 少し息が苦しくなるくらいの強い力で抱きしめられると、自分が存在していると実感する
48: 眩しい光よりも、間接照明や月明かりだけの空間にずっといたい
49: 誰かが自分のすぐ後ろに立っているという「気配」だけで、背筋が微かに粟立つ
50: すべてが終わった後の、感情の波が引いていく静かで虚無的な時間を愛している
```
