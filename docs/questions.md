## Phase 2：シチュエーション選択（全6問）

### 形式
4つの選択肢から、自分の心が最も動くものを選ぶ。

### Question 1: 契約の夜
**設定**: 白紙の契約書に書くルール

- **選択肢A**: すべての決定権を譲渡する
  - *判定*: Submission
- **選択肢B**: 許可なく指一本動かすな
  - *判定*: Dominance
- **選択肢C**: 恥ずかしい秘密を共有する
  - *判定*: Psychological
- **選択肢D**: セーフワードを設定する
  - *判定*: Trust / Care

---

### Question 2: 視界の境界線
**設定**: シルクの目隠しを使う際の役割

- **選択肢A**: 相手の目を隠して観察する
  - *判定*: Sadism / Voyeurism
- **選択肢B**: 相手に目隠しをされるのを待つ
  - *判定*: Submission / Restraint
- **選択肢C**: 隙間から覗き見る
  - *判定*: Psychological / Rebellion
- **選択肢D**: 目隠しをして耳元で囁き焦らす
  - *判定*: Dominance / Psychological

---

### Question 3: 完全なる拘束
**設定**: 椅子に固定された絶望的な状況での感情

- **選択肢A**: 圧倒的な安堵感と陶酔
  - *判定*: Bondage / Submission
- **選択肢B**: 抵抗して相手のS気を煽る
  - *判定*: Brat / Masochism
- **選択肢C**: 自分は縛る側（確認する側）だ
  - *判定*: Dominance
- **選択肢D**: 無様な姿を鏡で見せられる興奮
  - *判定*: Exhibitionism

---

### Question 4: 美しき罰
**設定**: ルールを破った際の「罰」の時間のスリル

- **選択肢A**: 冷徹な計算で相手を躾け直す
  - *判定*: Sadism / Discipline
- **選択肢B**: 恐怖に震えながら許しを乞う
  - *判定*: Masochism / Submission
- **選択肢C**: 痛みが来る直前の焦らし
  - *判定*: Psychological
- **選択肢D**: 屈辱的な言葉を言わせる
  - *判定*: Psychological Sadism

---

### Question 5: 境界線の融解
**設定**: 肌に残った「痕跡（傷や熱）」の意味

- **選択肢A**: 所有権の証明
  - *判定*: Dominance
- **選択肢B**: 支配されている証
  - *判定*: Submission / Masochism
- **選択肢C**: 感覚が真っ白になる鍵
  - *判定*: Extreme Masochism
- **選択肢D**: 耐性の限界を測る実験データ
  - *判定*: Analytical Sadism

---

### Question 6: 静寂の帰還
**設定**: プレイが終わった後の「アフターケア」

- **選択肢A**: 抱きしめられ肯定されたい
  - *判定*: Submissive Aftercare
- **選択肢B**: 相手の尊厳を整えてあげたい

- **選択肢D**: 冷静に振り返り次を話し合いたい
  - *判定*: Analytical

## Phase 3：深層心理の投影（自由記述・全2問）

### Question 1: 暗闇の囁き

**質問文**:
> 「拘束され視界を奪われたあなたの耳元で、誰かが『ある一言』を囁きました。  
> その言葉は何ですか？また、心と体はどう反応しましたか？」

**分析ポイント**:
- 囁かれた言葉の内容（命令系 / 賞賛系 / 所有系 / 脅迫系）
- 身体反応の記述（恐怖 / 安堵 / 興奮 / 抵抗）
- 言語化の精密さ（具体的 / 抽象的）

---

### Question 2: 絶対的な空白

**質問文**:
> 「目の前で完璧な存在がひざまずいています。  
> その人に対し、最初に『どんな言葉』をかけ、『どこに』触れますか？」

**分析ポイント**:
- 最初の言葉（優しさ / 命令 / 確認 / 試し）
- 触れる部位（頭 / 顎 / 首 / 手 / 触れない）
- 支配欲と配慮のバランス

## スコアリング設計（参考）

### Phase 1の集計方法
各質問の「惹かれる（右スワイプ）」に対して、以下の軸でポイントを加算：

- **D軸（Dominance）**: 8, 15, 17, 20, 25, 34, 48, 49
- **S軸（Submission）**: 2, 3, 5, 7, 12, 14, 16, 26
- **B軸（Bondage）**: 1, 11, 21, 23, 30, 39, 45
- **M軸（Masochism）**: 13, 26, 27, 31, 35, 38, 42
- **Psychological軸**: 6, 9, 19, 25, 28, 29, 33, 34, 35, 49
- **Sensory軸**: 10, 22, 24, 37, 40
- **Voyeurism/Exhibitionism軸**: 17, 19, 28, 41, 43, 46

### Phase 2の重み付け
各選択肢に対して、Phase 1の2倍の重みでスコアを加算。

### Phase 3のテキスト解析
AI（Gemini API）が以下を抽出：
- 支配欲/服従欲の強度
- 言語表現の繊細さ
- 配慮と暴力性のバランス
- 五感への言及
