## Phase III — 深層心理の投影（自由記述）

Phase1/2のスコアから傾向（dominant / submissive / switch）を自動判定し、対応する質問2問を表示。

```

[title]                                ← Cinzel / 薄グレー
[prompt テキスト]                      ← Shippori Mincho / lineHeight:2
                                          whiteSpace:pre-line（段落区切りあり）

┌─────────────────────────────────┐
│                                 │
│  [自由記述テキストエリア]         │  ← 最小20文字 / 最大500文字
│  placeholder: 耽美な例文         │    フォーカス時: グロウ（傾向カラー）
│                                 │
└─────────────────────────────────┘
あとXX文字 / XX / 500

[ 次へ / 解析へ進む ]               ← 20文字達成で活性化
```

傾向別 質問タイトル / プロンプト:
- **dominant** → 「支配者の眼差し」視界が開いた瞬間、完璧な存在への最初の言葉と接触
- **submissive** → 「深淵からの告白」拘束された暗闇で耳元に囁かれた一言
- **switch** → 「合わせ鏡の共犯者」対等な視線、銀トレイのリボンと南京錠
